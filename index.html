Photo-Sharpener

This project aims to enhance image quality by removing blur and imperfections. It evolved through multiple stages, from a simple CNN to a U-Net-like encoder-decoder and finally a ResUNet with residual blocks. Transfer learning with VGG19 provided the best results. The model works best on images around 480×584 px and demonstrates effective sharpening, though it may occasionally misinterpret colors. The main goal of the project was learning and experimentation rather than achieving a perfect solution.

Tech Stack: TensorFlow, OpenCV, Jupyter Lab, CUDA

Cflat

CFlat (C♭) is an interpreted programming language specifically designed for music composition. It allows you to write code that generates musical pieces, from simple melodies to complex polyphonic compositions. CFlat combines programming constructs with musical notation, enabling musicians and programmers to create music through code. The language provides intuitive syntax for musical elements like notes, chords, and tracks, along with programming features like variables, functions, and control structures.

Tech Stack: Java, Maven, ANTL4, MIDI

Good vibezz

The main goal of the project is to practice training and working with machine learning models, mainly the transformer architecture. The news is automatically fetched every day from newsapi.ai using a scheduled cron job. Newsapi.org was also used to acquire training data for the model Next, the articles are classified using a fine-tuned version of the BERT NLP model with a custom regression head added on top (predicting news sentiment in the [-1, 1] range).

Tech Stack: Python, PyTorch, Pandas, Transformers, Flask
